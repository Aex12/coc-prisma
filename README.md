# coc-prisma

Coc extension that implements Prisma Language Server

## Install

`:CocInstall coc-prisma`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-prisma-keymap)`

## Lists

`:CocList demo_list`
